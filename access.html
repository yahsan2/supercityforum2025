交通アクセス

🚅

電車でお越しの方

東京方面から

  1. 東京駅 → 金沢駅（北陸新幹線 約2時間30分）
  2. 金沢駅 → 加賀温泉駅（JR北陸本線 特急 約25分）
  3. 加賀温泉駅 → 会場（タクシー 約15分）

所要時間：約3時間10分

大阪方面から

  1. 大阪駅 → 金沢駅（JR特急サンダーバード 約2時間30分）
  2. 金沢駅 → 加賀温泉駅（JR北陸本線 特急 約25分）
  3. 加賀温泉駅 → 会場（タクシー 約15分）

所要時間：約3時間10分

名古屋方面から

  1. 名古屋駅 → 米原駅（JR東海道新幹線 約30分）
  2. 米原駅 → 加賀温泉駅（JR特急しらさぎ 約1時間40分）
  3. 加賀温泉駅 → 会場（タクシー 約15分）

所要時間：約2時間25分
